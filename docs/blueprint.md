# **App Name**: EnergyGlimpse

## Core Features:

- Real-time Data Fetch: Continuously fetch energy consumption data from your Supabase database.
- Consumption Display: Display current energy consumption in a clear, numerical format.
- Historical Data View: Visualize historical consumption data using interactive charts (e.g., daily, weekly, monthly).
- Cost Estimation: Calculate estimated energy costs based on current consumption and billing rates.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to represent the technological and monitoring aspect of the app.
- Background color: Light gray (#EEEEEE) to provide a clean and neutral backdrop for the data.
- Accent color: Electric blue (#03A9F4) to highlight important data and interactive elements.
- Body and headline font: 'Inter' (sans-serif) for clear readability on data and labels.
- Use minimalist line icons to represent different data types and functionalities.
- Use a card-based layout to present data in organized and digestible sections.
- Subtle transitions when updating data displays for a smooth user experience.
